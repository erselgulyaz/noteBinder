Say Hello to noteBinder :)

noteBinder is real time push notification plugin. If you are using noteBinder; form sending, after window load, click, change etc. form actions user friendly smaller message push on user window.

noteBinder is customizable your claims. For example; message alignment is right and top with plugin default options. If you are change verticalPos and horizontalPos, change this option and message alignment make left and bottom.

See other options

Installation

noteBinder installation and customizing is very easy. First one DOWNLOAD with github link and export plugin files.

				
					<link rel="stylesheet" href="../src/css/noteBinder.css" />
				
				

				
					<script src="../src/js/noteBinder.js"></script>

And calling plugin initialization.

				
					noteBinder.init();

Options

Doesn't my default properties fit you? noteBinder options are customizable your claims :) You are calling noteBinder init function create one variable and send to options. For example:

Name Default Description
openEffect fade Message opened effect. Parameters: fade,bounce,slideToVer,slideToHor
closeEffect fade Message closed effect. Parameters: fade,bounce,slideToVer,slideToHor
maxItem 3 Message length on screen. Use it integer parameters
waiting 5000 Message second parameter for visibility.
message hi! This is message content. Your change this text or add html code blocks.
verticalPos top Message vertical alignment for screen size. Other parameters: top , center , bottom
verticalSpace 32 message spaces for your verticalPos parameters. For example: you use it verticalPos: top , message top space is 32px
horizontalPos right Message horizontal alignment for screen size. Other parameters: left , center , right
horizontalSpace 32 message spaces for your horizontalPos parameters. For example: you use it horizontalPos: right , message right space is 32px
clickClose false If you are set true parameters, find your message template closeElement parameter. And click for this class your message is removed.
wrapClass noteBinder Your message wrapped div on use this class.
closeElement false If you are set ".close" parameters, find your message template "close" class. And your message is removed click for this class. But first set true parameter "clickClose" options.
posForGroup before This option for new messages. For example set parameter is "before", New message added top positions for other messages. This option see the verticalPos elements. Other parameters: after

How to Use

First one create one variable

				
					var options;

Then look at the Options section. Because all parameters and contents writing this section. Copy the properties you want to change and append to variables. 

				
					var options = {
						verticalPos: "center",
						horizontalPos: "center",
						posForGroup: "after"
					}

Adding new features you select a call noteBinder 

				
					noteBinder.init(options);
